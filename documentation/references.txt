	- used in DairyFarmerClient.java
 * gson-javatime-serialisers by gkopff
	- https://github.com/gkopff/gson-javatime-serialisers
	- https://mvnrepository.com/artifact/com.fatboyindustrial.gson-javatime-serialisers/gson-javatime-serialisers/1.1.1
	- used for serializing and deserializing Java's LocalData and LocalTime
	- used in DairyFarmerClient.java

	- https://www.eclipse.org/efxclipse/install.html
	- UI library for design elements
	- used in CalendarUI.fxml

	- http://gluonhq.com/products/scene-builder/
	- UI for utilizing e(fx)clipse
	- used in CalendarUI.fxml
	- used in CreateEventUI.fxml
